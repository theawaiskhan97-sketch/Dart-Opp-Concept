# Dart OOP Concepts

This repository contains my practice programs and notes while learning Object-Oriented Programming (OOP) in Dart.

## Topics Covered

### 1. Class & Object

* Creating classes
* Creating objects
* Accessing properties and methods

### 2. Constructor

* Default constructors
* Parameterized constructors
* Initializing object data

### 3. Named Constructor

* Creating multiple constructors
* Providing custom ways to create objects

### 4. Encapsulation

* Private variables
* Getters and setters
* Data protection

### 5. Inheritance

* Reusing code from parent classes
* Creating child classes using `extends`

### 6. Method Overriding

* Modifying inherited behavior
* Using the `@override` annotation

### 7. Abstract Classes

* Creating blueprints for subclasses
* Defining abstract methods

### 8. Interfaces

* Implementing contracts using `implements`
* Enforcing method definitions

### 9. Polymorphism

* Using parent references for child objects
* Runtime method selection

### 10. Mixins

* Reusing functionality without inheritance
* Using the `with` keyword

## Repository Purpose

The purpose of this repository is to:

* Learn Dart OOP fundamentals
* Practice object-oriented programming concepts
* Build a strong foundation for Flutter development
* Prepare for real-world application development
